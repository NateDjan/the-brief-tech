Il y a deux ans, Cognition lançait **Devin** et promettait le premier ingénieur logiciel IA autonome. Beaucoup étaient sceptiques. Fin mai 2026, le doute n'est plus permis : la startup américaine vient de lever **plus d'un milliard de dollars** à une valorisation **pré-money de 25 milliards de dollars** (26 milliards post-money). Une opération qui positionne Cognition parmi les startups les plus valorisées de l'histoire de l'IA.

## Devin : deux ans après le lancement, le bilan

Quand Cognition a lancé Devin en 2024, la promesse était radicale : un agent IA capable de **planifier, écrire, debugger et déployer du code** de manière autonome, comme un vrai développeur junior. Les démonstrations étaient impressionnantes — mais aussi controversées, certains experts reprochant des benchmarks gonflés.

Deux ans plus tard, les chiffres parlent d'eux-mêmes :

- **Croissance de l'usage entreprise** : plus de **10x depuis le début de 2026**
- **Revenu annuel récurrent** : **492 millions de dollars** (run-rate)
- **Croissance mensuelle de l'usage** : **+50 % chaque mois** pendant les 6 derniers mois

Ces métriques sont exceptionnelles pour une startup de cette taille et de cet âge. Elles expliquent en grande partie la valorisation stratosphérique de 26 milliards.

## Ce que Devin fait vraiment

Devin n'est pas un simple assistant de code comme GitHub Copilot. C'est un **agent autonome** : il dispose d'un terminal, d'un navigateur, d'un éditeur de code et peut **exécuter des tâches de bout en bout** sans intervention humaine.

### Cas d'usage concrets

1. **Refactoring de codebase** : Devin peut analyser des milliers de lignes de code et proposer (puis exécuter) une restructuration complète
2. **Correction de bugs** : à partir d'un rapport d'erreur, Devin identifie la cause, écrit le fix et crée la pull request
3. **Développement de features** : sur la base d'une spec en langage naturel, Devin code, teste et documente une nouvelle fonctionnalité
4. **Migrations de stack** : changer de framework, migrer une API, passer de Python 3.10 à 3.12 — Devin peut gérer ces migrations fastidieuses

### La différence avec GitHub Copilot et Cursor

GitHub Copilot et Cursor sont des outils d'**assistance à la complétion de code** — ils suggèrent, vous décidez. Devin est un **agent** : il prend des décisions, exécute du code, itère sur ses erreurs. La différence est fondamentale dans la valeur créée pour les entreprises.

C'est pourquoi Cognition vise d'abord le **marché enterprise** : les grandes entreprises avec des milliers de tickets d'ingénierie en retard, des migrations de legacy systems interminables, des équipes débordées. Devin ne remplace pas un CTO senior — il absorbe le travail routinier que les bons ingénieurs détestent.

## L'impact sur le marché du travail tech

La levée de Cognition relance un débat que beaucoup préfèrent éviter : **les développeurs sont-ils menacés ?**

### La réponse nuancée de l'industrie

Les experts sont divisés. Certains, comme les équipes de Cognition elles-mêmes, soutiennent que Devin **augmente** les développeurs sans les remplacer — il leur permet de se concentrer sur les problèmes complexes tandis qu'il gère le "toil" (travail répétitif).

D'autres, notamment les syndicats de l'industrie tech et certains économistes du travail, s'inquiètent d'une **substitution progressive**. Avec 492 millions de dollars de revenu annualisé, Devin remplace déjà un volume de travail d'ingénierie conséquent.

### Les chiffres qui font réfléchir

Si Devin facture en moyenne (hypothèse) 500 à 1 000 $/mois par siège enterprise, 492 M$ de ARR représente entre **40 000 et 80 000 "développeurs équivalents"** que les entreprises n'ont pas eu besoin d'embaucher ou qu'elles ont pu ne pas renouveler.

Ce n'est pas négligeable — mais c'est aussi, pour l'instant, une fraction infime du marché mondial du développement logiciel, estimé à plusieurs millions de postes.

## La concurrence : qui veut aussi être "l'agent dev" numéro un ?

Cognition n'est pas seule sur ce marché. Les agents développeurs sont l'un des segments les plus compétitifs de l'IA en 2026 :

- **GitHub Copilot Workspace** (Microsoft/OpenAI) — l'agent autonome de GitHub, avec 177 millions d'utilisateurs potentiels
- **Cursor** — l'éditeur IA qui a conquis les développeurs avec son interface intuitive
- **Replit Agent** — pour le développement rapide d'apps et de scripts
- **Sweep** — spécialisé dans les issues GitHub automatisées
- **SWE-agent** (Stanford) — open source, taillé pour la recherche

La valorisation de 26 milliards suggère que les investisseurs croient que Cognition a une longueur d'avance significative — notamment sur la qualité d'exécution autonome des tâches complexes.

## Quobly, Modal, Exa : le contexte d'une vague de méga-tours

La levée de Cognition s'inscrit dans une tendance plus large : depuis début 2026, les méga-tours pour les startups IA se multiplient :

- **Exa** (moteur de recherche IA) : 250 M$ — a16z
- **Modal** (infrastructure IA) : 355 M$ — General Catalyst
- **TensorWave** (clouds IA) : 300 M$ — Nvidia Ventures
- **Generalist AI** : 400 M$ — Radical Ventures

L'argent afflue vers les startups qui résolvent des problèmes concrets avec de l'IA — et Devin, avec ses 492 M$ de ARR, coche toutes les cases.
